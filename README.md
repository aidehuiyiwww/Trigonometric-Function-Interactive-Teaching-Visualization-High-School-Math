## 🔎 Project Overview / 项目概述

This is a lightweight, zero-configuration front-end visualization project designed specifically for high school trigonometric function teaching. Based on native HTML, CSS and Canvas, the system supports real-time rendering of sine, cosine and tangent functions. Users can dynamically adjust four core parameters: amplitude, angular frequency, initial phase and vertical offset. Equipped with five-point plotting marking, auxiliary reference lines, dual-image contrast, transformation animation, full-screen teaching mode and one-click screenshot function, it fully covers all key and difficult points of trigonometric graph transformation in high school curriculum.


本项目是专为高中数学三角函数教学打造的轻量化、零配置前端可视化工具。基于原生HTML、CSS和Canvas开发，支持正弦、余弦、正切函数实时绘制。用户可动态调节振幅、角频率、初相、上下平移四大核心参数。内置五点作图标注、辅助参考线、双图像对比、变换动画、全屏教学模式和一键截图功能，完整覆盖高中三角函数图像变换全部重难点。

## ✨ Core Features / 核心功能

- Real-time rendering of `sin(x)`, `cos(x)` and `tan(x)` functions
- Dynamic adjustment of **A, ω, φ, k** parameters with millisecond-level graph refresh
- Standard coordinate system with π-scale grid lines and automatic tangent asymptote rendering
- **Five-point plotting** marking (core high school exam point)
- Auxiliary reference lines and original function contrast display
- Vivid dynamic animations: phase shift, period scaling, amplitude scaling
- Full-screen classroom demonstration mode
- One-click screenshot saving for courseware production
- Zero environment dependency, double-click to run directly

- 实时绘制正弦、余弦、正切三类基础三角函数图像
- 四大核心参数滑块实时调节，图像毫秒级刷新
- 标准π刻度坐标系，正切函数渐近线自动渲染
- 高考必考**五点作图法**关键点坐标标注
- 辅助参考线与原始函数对比叠加显示
- 生动的动态演示动画：相位平移、周期伸缩、振幅伸缩
- 课堂全屏演示模式
- 一键截图保存，方便制作课件
- 无需配置任何环境，双击直接运行
